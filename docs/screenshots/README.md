# Скриншоты для README

Положи сюда PNG-скрины приложения с такими именами — они подтянутся в корневой `README.md`:

| Файл            | Что снять                                              |
| --------------- | ----------------------------------------------------- |
| `feed.png`      | Лента с постами (видны вложения, лайки, комментарии)  |
| `profile.png`   | Страница профиля (аватар, обложка, фото, «о себе»)    |
| `messages.png`  | Мессенджер: диалог с вложением/голосовым              |
| `call.png`      | Активный звонок (WebRTC, панель с видео)              |
| `channel.png`   | Страница канала (картинка, бейдж роли, участники)     |
| `games.png`     | Раздел игр (или одна из игр в процессе)               |

Как снять на Windows: `Win + Shift + S` → выдели область → сохрани в эту папку под нужным именем.
Рекомендуемая ширина — 1200–1600px, формат PNG.
